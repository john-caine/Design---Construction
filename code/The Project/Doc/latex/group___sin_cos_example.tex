\begin{DoxyParagraph}{}
Mathematically, the Pythagorean trignometric identity is defined by the following equation\-: 
\begin{DoxyPre}sin(x) * sin(x) + cos(x) * cos(x) = 1\end{DoxyPre}
 where {\ttfamily x} is the angle in radians.
