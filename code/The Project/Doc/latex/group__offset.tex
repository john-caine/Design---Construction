\subsection{Detailed Description}
Adds a constant offset to each element of a vector.


\begin{DoxyPre}   
    pDst[n] = pSrc[n] + offset,   0 <= n < blockSize.   
\end{DoxyPre}


There are separate functions for floating-\/point, Q7, Q15, and Q31 data types.
